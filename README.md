# cable_ready.cr

This is a Crystal port of the [CableReady](https://github.com/hopsoft/cable_ready) Ruby library.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     cable_ready:
       github: marcoroth/cable_ready
   ```

2. Run `shards install`
